# About Shaik - Professional Portfolio

![Portfolio Preview](https://via.placeholder.com/800x400?text=Portfolio+Preview)

A professional portfolio website for **Shaik Mohammed Asif**, showcasing skills, projects, and professional background. Built with modern web technologies to ensure a responsive, accessible, and visually appealing experience.

## 🚀 Live Demo
https://asif-shaik-10.github.io/about-shaik/

## 🛠️ Tech Stack
- **Frontend Framework**: React (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: Vercel/Netlify (Recommended)

## ✨ Features
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Modern UI**: Glassmorphism effects, smooth gradients, and interactive animations.
- **Dark Mode Aesthetic**: Professional dark theme with indigo accents.
- **Sections**:
  - **Home**: Dynamic introduction with Resume download.
  - **About**: Professional summary and education details.
  - **Skills**: Visual representation of technical expertise.
  - **Projects**: Showcase of key projects with GitHub links.
  - **Contact**: Easy access to social links and email.

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/asif-shaik-10/about-shaik.git
   cd about-shaik
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start local development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
